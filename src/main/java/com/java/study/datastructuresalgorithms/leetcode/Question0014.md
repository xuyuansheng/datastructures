###14.最长公共前缀
&emsp;&emsp;编写一个函数来查找字符串数组中的最长公共前缀。
如果不存在公共前缀，返回空字符串 ""。

示例 1:  
输入: ["flower","flow","flight"]  
输出: "fl"  

示例 2:  
输入: ["dog","racecar","car"]  
输出: ""  
解释: 输入不存在公共前缀。
  
说明:  
所有输入只包含小写字母 a-z 。 

思路: 从左到右逐个字符顺序遍历所有字符串,当一列中所有字符都一样时,就继续下一列,直到遍历完其中任何一个字符串
或者出现了不一致的字符为止,那之前的字符顺序连接起来就是最长前缀子串
